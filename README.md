# Async Library Management System


# Objectives
- Implement an asynchronous system using Python asyncio
- Allow multiple users to borrow and return books concurrently
- Simulate real-world delays using `asyncio.sleep()`
- Prevent data conflicts using locks



# Tools and Technologies
- Python 3
- asyncio (Asynchronous programming)
- datetime module
- typing (for structured data)



# Features
- Borrow books (multi-user support)
- Return books with fine calculation
- Async delay simulation (1 second)
- Safe data handling using `asyncio.Lock()`



# How to Run the Project

1. Make sure Python is installed (Python 3.8+ recommended)

2. Save the file as: library_system.py
